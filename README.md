# 🧮 Numerical Base Calculator

## 📌 Sobre o projeto

O **Numerical Base Calculator** é uma calculadora desenvolvida em C++ para disciplina de **computação numérica**, capaz de realizar **conversão entre diferentes bases numéricas** e executar **operações matemáticas (soma e multiplicação)** em diferentes sistemas de numeração.

Este projeto foi desenvolvido com o objetivo de reforçar conceitos fundamentais como:

* Representação de números em diferentes bases
* Manipulação de strings numéricas
* Lógica de algoritmos matemáticos

---

## 🛠️ Tecnologias utilizadas

* 💻 **C++**
* ⚙️ Compilação via terminal (g++, clang, etc)

---

## 💡 Funcionalidades

* 🔄 Conversão entre bases numéricas (ex: binário, decimal, hexadecimal)
* ➕ Soma de números em diferentes bases
* ✖️ Multiplicação de números em diferentes bases
* 📊 Manipulação de valores não decimais

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram aplicados conceitos importantes como:

* Algoritmos de conversão de bases
* Estruturação de lógica matemática em código
* Manipulação de entrada e saída em C++
* Pensamento computacional aplicado à matemática
