## 📌 Sobre o projeto

O **Numerical Base Calculator** é uma calculadora desenvolvida em C++ para disciplina de **computação numérica**, capaz de realizar **conversão entre diferentes bases numéricas** e executar **operações matemáticas (soma e multiplicação)** em diferentes sistemas de numeração.

Este projeto foi desenvolvido com o objetivo de reforçar conceitos fundamentais como:

* Representação de números em diferentes bases
* Manipulação de strings numéricas
* Lógica de algoritmos matemáticos

---

## 📌 About the Project

The **Numerical Base Calculator** is a calculator developed in C++ for a **numerical computing** course, capable of performing **conversion between different number bases** and executing **mathematical operations (addition and multiplication)** across various numeral systems.

This project was developed to reinforce fundamental concepts such as:

* Representation of numbers in different bases
* Manipulation of numeric strings
* Mathematical algorithm logic
